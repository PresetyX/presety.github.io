Projetos em Destaque

TechStore - Plataforma de E-commerce Completa

Uma solução de e-commerce moderna e totalmente responsiva, construída do zero para oferecer uma experiência de compra rápida e intuitiva. O front-end, desenvolvido em React, proporciona uma navegação fluida e dinâmica, enquanto a robusta API RESTful em Node.js com Express garante a segurança e o gerenciamento eficiente de produtos, usuários e pedidos. O sistema inclui autenticação de usuários com JWT, carrinho de compras, integração com gateway de pagamento e um painel administrativo exclusivo para gestão de estoque e análise de vendas.

React.js, Node.js, Express, MongoDB, JWT (JSON Web Tokens), CSS-in-JS, API RESTful.

Data Analytics Automator

Um script de automação em Python projetado para otimizar a tomada de decisões de negócio. A ferramenta coleta dados brutos de múltiplas fontes (APIs externas e arquivos .CSV), realiza a limpeza e transformação dos dados utilizando a biblioteca Pandas, e executa análises estatísticas para identificar tendências e padrões. Ao final do processo, gera e envia automaticamente relatórios visualmente claros em PDF por e-mail para os stakeholders, economizando dezenas de horas de trabalho manual por semana.

Tecnologias: Python, Pandas, Matplotlib, smtplib, Requests, APIs REST.

Guia de Filmes Mobile

Um aplicativo mobile para iOS e Android, desenvolvido com React Native, que serve como um guia de bolso para entusiastas de cinema. O app consome a API do The Movie Database (TMDb) para buscar e exibir informações em tempo real sobre filmes em cartaz, populares e clássicos. A interface é limpa e focada na usabilidade, com funcionalidades de busca, listas de favoritos e detalhes de filmes. O uso do React Native permitiu a criação de um app de alta performance para ambas as plataformas a partir de um único código-base.

Tecnologias: React Native, JavaScript (ES6+), Axios, The Movie Database (TMDb) API, State Management.
